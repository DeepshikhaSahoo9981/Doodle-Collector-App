Doodle-Collector-App

An interactive mobile application designed to break creative blocks. It utilizes Gemini 2.5 Flash to spark tailored sketching prompts and handles automated, un-throttled asset synchronization directly to isolated Google Cloud folder environments.

🤖 AI Orchestration: Integrated the Gemini 2.5 Flash API to dynamically output lightweight, context-aware design concepts.

☁️ Cloud Synchronization: Designed an automated backend proxy layer to resolve unauthenticated access restrictions, programmatically mutating file permission models to allow fluid asset rendering.

⚡ UI Performance: Engineered a custom layout constraint grid in React Native that handles asynchronous layout updates locally, preventing UI thread freezing during high-resolution cloud image fetches.
